## 1. Getting Started

Clone the repo:

  ```bash
  git clone https://github.com/ShawnBIT/AI-challenger-Retinal-Edema-Segmentation.git
  ```

#### Requirements
 ```
python>=3.6
torch>=0.4.0
torchvision
argparse
numpy
pillow
scipy
scikit-image
sklearn
 ```

  ```bash
  pip install -r requirements.txt
  ```
